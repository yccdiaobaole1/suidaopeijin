# ANSYS APDL 隧道衬砌建模专家

生成或修复 ANSYS APDL 隧道衬砌有限元分析命令流，用于计算衬砌内力（弯矩、轴力、剪力）。

## 功能

- 生成完整的 APDL 命令流（从几何建模到内力提取）
- 修复现有 APDL 代码的几何、荷载、网格问题
- 优化预制块分割策略（A型块分割、B型块整体）
- 调整材料参数、荷载工况、边界条件
- 解决常见错误（几何警告、内力为零、刚度翻倍等）

## 使用方法

```
/apdl-tunnel [任务描述]
```

**示例：**
- `/apdl-tunnel 生成双线铁路隧道命令流，衬砌厚度0.7m，围岩等级IV级`
- `/apdl-tunnel 修复B型预制块分割问题，保持整体不分块`
- `/apdl-tunnel 调整荷载：顶部174.4kPa，侧向29.9-68.1kPa梯形分布`
- `/apdl-tunnel 提取拱顶、拱腰、拱脚的弯矩和轴力`

## 核心原则

### 1. 六阶段标准流程

```apdl
/clear
/prep7
! ========== 阶段1：前处理与几何建模 ==========
! 关键点定义、圆弧绘制、面生成

! ========== 阶段2：材料与单元定义 ==========
! 混凝土参数、弹簧刚度、单元类型

! ========== 阶段3：网格划分 ==========
! 线网格、面网格、映射网格

! ========== 阶段4：荷载施加 ==========
! 重力、非均匀水土压力循环

! ========== 阶段5：边界条件与求解 ==========
/SOLU
! 约束、弹簧生成、SOLVE

! ========== 阶段6：后处理与内力提取 ==========
/POST1
SET, LAST  ! 关键：必须读取求解结果
! 提取弯矩、轴力、剪力
```

### 2. 几何建模精髓

**关键点分层定义：**
```apdl
! 中心线关键点 (K1-K9) - 用于提取内力
K, 1, 0.00000, -4.52400, 0
K, 2, 4.89643, -3.85960, 0
...

! 内轮廓关键点 (K11-K19) - 衬砌内侧
K, 11, 0.00000, -4.17400, 0
...

! 外轮廓关键点 (K21-K29) - 衬砌外侧
K, 21, 0.00000, -4.87400, 0
...

! 圆弧中间点 (K41-K59) - 仅用于画弧，不作为分块点
K, 41, 2.47000, -4.35700, 0
...
```

**三点画弧法（避免几何警告）：**
```apdl
LARC, K1, K2, K_mid    ! 不指定半径，让ANSYS自动贴合
```
- 坐标精度：关键点6位小数，中间点5位小数
- 避免 "Arc of radius X not recommended" 警告

**预制块分割原则（关键！）：**
- **A型预制块**：上半拱60°三等分（3块）
  - 在拼缝处创建关键点（如K4, K5, K6）
  - 创建径向切割线连接内外轮廓
- **B型预制块**：保持整体，不在中间分块
  - 即使由两个不同圆心的圆弧组成（如18.4°上段+下段）
  - 中间点K43仅用于画弧，不创建径向切割线
  - 直接从墙角相切点连接到A/B拼缝点

### 3. 材料参数标准

**混凝土参数：**
```apdl
E_C = 34.5e9      ! 弹性模量 (Pa) - C30混凝土
U_C = 0.2         ! 泊松比
DENS_C = 2420     ! 密度 (kg/m³)
TH = 0.7          ! 衬砌厚度 (m)
```

**围岩弹簧刚度（根据地层等级）：**
```apdl
KS = 400e6        ! II-III级围岩（好）
KS = 200e6        ! IV级围岩（中等）
KS = 143.8e6      ! IV-V级围岩（差）
KS = 100e6        ! V-VI级围岩（很差）
```

**单元类型：**
```apdl
ET,1,BEAM188,,,3       ! 虚拟荷载梁（外轮廓）
ET,2,PLANE182          ! 二维实体混凝土面
ET,3,BEAM188           ! 真实内力梁（中心线）
ET,4,LINK10,,,1        ! 弹簧单元（围岩反力）
```

### 4. 荷载施加算法

**非均匀水土压力循环（梯形分布）：**
```apdl
! 定义荷载基准值
*SET, Q_TOP, 174400*BL     ! 顶部垂直压力 (Pa)
*SET, Q_BOT, 174400*BL     ! 底部垂直压力 (Pa)
*SET, E_TOP, 29900*BL      ! 顶部水平侧压 (Pa)
*SET, E_BOT, 68100*BL      ! 底部水平侧压 (Pa)

! 循环施加非均匀压力
*GET, N_COUNT, NODE, , COUNT
*DO, I, 1, N_COUNT
  *GET, NID, NODE, I, NUM, MIN
  *GET, Y_I, NODE, NID, LOC, Y
  *GET, X_I, NODE, NID, LOC, X

  ! 根据Y坐标线性插值计算压力
  *IF, Y_I, GT, Y_TOP, THEN
    Q_I = Q_TOP
  *ELSEIF, Y_I, LT, Y_BOT, THEN
    Q_I = Q_BOT
  *ELSE
    Q_I = Q_TOP + (Q_BOT - Q_TOP) * (Y_TOP - Y_I) / (Y_TOP - Y_BOT)
  *ENDIF

  ! 施加节点力
  F, NID, FY, -Q_I
*ENDDO
```

**关键注意事项：**
- 使用精确的Y坐标范围判断
- 避免浮点数判断失效（不要用 `Y_I, EQ, Y_TOP`）
- 荷载单位统一为 Pa（帕斯卡）

### 5. 网格划分策略

**映射网格（推荐）：**
```apdl
MSHAPE, 0, 2D      ! 2D四边形网格
MSHKEY, 1          ! 启用映射网格
LESIZE, ALL, 0.2   ! 单元尺寸0.2m

! 分别控制不同线段的网格密度
LSEL, S, LINE, , 8, 14    ! 中心线
LESIZE, ALL, 0.2
LSEL, S, LINE, , 15, 21   ! 内轮廓
LESIZE, ALL, 0.2
LSEL, S, LINE, , 1, 7     ! 外轮廓
LESIZE, ALL, 0.2
LSEL, S, LINE, , 22, 35   ! 径向线
LESIZE, ALL, , , 1        ! 径向线只分1段
ALLSEL, ALL
```

**关键要求：**
- 使用规则四边形面（12-18个面）
- 径向切割线将截面分割成规则四边形
- 网格点与预制块拼缝对齐
- 单元尺寸：0.1-0.3m（根据精度要求）

### 6. 后处理内力提取

**完整提取命令：**
```apdl
/POST1
SET, LAST              ! 关键：必须读取最后一步结果

ALLSEL, ALL
ESEL, S, TYPE, , 3     ! 选择Type 3单元（真实内力梁）

! 定义内力表格
ETABLE, WJI, SMISC, 3   ! 弯矩 I端
ETABLE, WJJ, SMISC, 16  ! 弯矩 J端
ETABLE, JLI, SMISC, 6   ! 剪力 I端
ETABLE, JLJ, SMISC, 19  ! 剪力 J端
ETABLE, ZLI, SMISC, 1   ! 轴力 I端
ETABLE, ZLJ, SMISC, 14  ! 轴力 J端

! 绘制内力图
PLLS, WJI, WJJ, -1
/image, save, wanju_f, bmp

PLLS, ZLI, ZLJ
/image, save, zhouli_f, bmp

PLLS, JLI, JLJ
/image, save, jianli_f, bmp

! 输出内力表格
PRETAB, WJI, ZLI, JLI
```

**SMISC索引对应关系：**
- SMISC,1 = 轴力 I端 (N)
- SMISC,3 = 弯矩 I端 (N·m)
- SMISC,6 = 剪力 I端 (N)
- SMISC,14 = 轴力 J端 (N)
- SMISC,16 = 弯矩 J端 (N·m)
- SMISC,19 = 剪力 J端 (N)

## 常见问题速查表

| 问题 | 原因 | 解决方案 |
|------|------|--------|
| 圆弧几何警告 | 半径与点位精度不匹配 | 使用 `LARC, P1, P2, P_mid`，省略半径参数 |
| 网格划分失败 | 面定义不规则 | 确保四边形面，使用 `MSHKEY,1` 映射网格 |
| 内力结果为零 | 未读取求解结果 | 添加 `SET, LAST` 命令 |
| 弯矩突变 | 网格点不与拼缝对齐 | 按预制块分割创建关键点 |
| 刚度翻倍 | 实体和梁重复赋材料 | 虚拟梁用虚拟材料 `E=1` |
| 自重计算错误 | 实体和梁都计算密度 | 虚拟梁密度设为 `0` |
| 水土压力分布异常 | Y坐标判断范围错误 | 精确定义 `Y_TOP` 和 `Y_BOT` |
| B型预制块被分割 | 中间点作为分块点 | 仅用于画弧，不创建径向切割线 |
| PRETAB报错 | 未选择单元或未SET | 先 `ESEL,S,TYPE,,3` 再 `SET,LAST` |

## 代码生成规范

生成 APDL 代码时必须遵循：

1. **精度控制**：关键点坐标保留6位小数
2. **注释规范**：每个阶段添加分隔注释（用 `!` 和 `=`）
3. **变量命名**：使用工程习惯（R_TOP, E_C, KS, TH, BL）
4. **单位统一**：长度(m)、力(N)、压力(Pa)、密度(kg/m³)
5. **结构清晰**：严格按六阶段流程组织
6. **错误预防**：后处理必须添加 `SET, LAST`
7. **预制块原则**：A型块分割，B型块整体
8. **荷载精确**：使用循环算法施加非均匀压力

## 输出格式

生成的 APDL 命令流应包含：

1. **文件头注释**：说明用途、工况、日期
2. **参数定义区**：集中定义所有可调参数
3. **六阶段代码**：清晰分隔，注释完整
4. **关键检查点**：在关键步骤后添加 `ALLSEL,ALL`
5. **结果输出**：自动保存弯矩、轴力、剪力图片

## 与配筋优化系统集成

APDL 计算的内力结果将传递给配筋优化系统：

```
APDL内力计算 → 荷载包络 → NSGA-II主筋优化 → 拉筋设计 → 分布筋设计
```

确保输出单位：
- 弯矩：N·m
- 轴力：N
- 剪力：N
