## 问题建模

### 决策变量

x = [grade ∈ {HRB400, HRB500}, d ∈ {6,...,25}mm, s ∈ {100,...,250}mm]

### 目标函数

```
min f₁(x) = Σ(ρᵢ × Vᵢ × Pᵢ)     — 单位长度钢筋成本 (CNY/m)
min f₂(x) = max(wₖ)               — 最大裂缝宽度 (mm)
max f₃(x) = s                      — 主筋间距 (mm)
```

### 自适应目标策略

对于全截面受压的截面类型（B/C 型），裂缝宽度恒为零，
第二目标自动从裂缝宽度切换为最小安全系数（SF），
以保持三目标优化的区分度。

切换判据：首代种群中所有有效解的最大裂缝宽度 < 10⁻⁹ mm。

- 受影响截面类型: B/C
- 目标切换: f₂(x) = max(wₖ) → f₂(x) = -min(SF)

### 约束条件

```
s.t.  γ₀·S ≤ R(x)                 — 承载力极限状态
      wmax ≤ wlim = 0.2mm          — 裂缝宽度限值 (Q/CR 9129-2018)
      ρ ≥ ρmin                      — 最小配筋率
      smin ≤ s ≤ smax              — 间距构造要求
      c ≥ cmin                      — 保护层厚度
```
